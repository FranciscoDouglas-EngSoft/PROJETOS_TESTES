# Mão Animada

Este projeto é uma animação de uma mão que se move de forma autônoma utilizando HTML, CSS e JavaScript. Abaixo estão as instruções para configurar e executar o projeto.

## Estrutura do Projeto

```
mao-animada
├── src
│   ├── index.html          # Ponto de entrada da aplicação
│   ├── css
│   │   └── styles.css      # Estilos globais da aplicação
│   ├── js
│   │   └── main.js         # Lógica JavaScript para a animação
│   └── components
│       └── hand
│           ├── hand.html   # Estrutura HTML específica da mão
│           └── hand.css     # Estilos específicos para a mão
├── .gitignore              # Arquivos e pastas a serem ignorados pelo git
├── package.json            # Configuração do npm
└── README.md               # Documentação do projeto
```

## Como Executar

1. Clone o repositório:
   ```
   git clone <URL_DO_REPOSITORIO>
   ```

2. Navegue até o diretório do projeto:
   ```
   cd mao-animada
   ```

3. Instale as dependências (se houver):
   ```
   npm install
   ```

4. Abra o arquivo `src/index.html` em um navegador para visualizar a animação da mão.
